# Simple Calculator in C

This is a basic calculator program written in C.

## Features
- Menu-based system
- Addition, Subtraction, Multiplication, Division
- User input handling
- Typecasting for accurate division

## What I learned
- Taking input using scanf
- Using if-else conditions
- Debugging errors (like missing & in scanf)
- Basic program flow

## How to run
1. Compile:
   gcc main.c -o calc
2. Run:
   ./calc
